Plugin para insertar Scripts de Vivet

== Description ==
Este plugin te permite agregar a todas las entradas de tu WordPress un widget con acceso directo a Vivet.

== Installation ==
Simplemente accede por FTP a tu instalación de WordPress y en la carpeta wp-content/plugins copia este archivo. Si no existiera la carpeta plugins, créala.
